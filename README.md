# chinese-haomenguoji-pages

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件的形式存放，不针对任何特定域名或网站，仅作为页面资源的集中存储与展示。

## 目录说明

```
.
├── index.html          # 首页入口文件
├── pages/              # 存放各独立 HTML 页面
│   ├── example1.html
│   └── example2.html
└── assets/             # 公共资源目录（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：仓库的默认首页，可在此处导航到其他页面。
- `pages/`：所有归档的独立 HTML 页面均放置在此目录下。
- `assets/`：存放页面所需的公共资源文件，按类型分类。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，不依赖外部动态服务或数据库。
- 页面之间无强耦合关系，可按需新增或更新。
- 归档内容遵循一般网页规范，无特殊功能要求。

## 维护说明

- 仓库维护者可根据需要添加、修改或移除 `pages/` 下的页面文件。
- 新增页面时，建议同时更新 `index.html` 中的导航链接（如有）。
- 公共资源（CSS/JS/图片）请统一放置在 `assets/` 目录下，避免重复。
- 本仓库仅用于静态页面归档，不提供持续集成或自动部署配置。
